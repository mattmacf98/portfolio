eBay Scraper

The ebay scraper is a flask based web app which uses the BeautifulSoup library to scrape the products from ebay's deals of the day page. Then it allows the user to sort by price, percent off, and -- two not supported by ebay -- original price and money off.

CrimeTracker

The CrimeTracker app is an Android app that pulls crime data from the BPD and displays it on an intuitive map with pins or heat map. The features included are: 911 calling, ICE calling, Danger zone alerts, viewing individual crime details, and seeing a heatmap of all the crimes in the Baltimore area.

Song Genre Classifier

The Song Genre Classifier is a machine learning model built on keras. It takes in word emmbeddings representing lyrics of a song and will classify if the song is a: Rock, Hip-hop, Pop, Country, or Metal song. It is an ensamble of an RNN and a CNN and achieved an accuracy of 61% over about 40,000 song titles.

Tonizer

Tonizer is a W.I.P., it uses FFT to estimate frequency of whistles and then returns the user their tune in notes which the user can modify and listen to.

PCA Imagizer

PCA Imagizer is a W.I.P., it uses k-means clustering to find the k components-- the k average colors-- of the submitted image, it then shows the user a side-by-side of the original image and the modified one.
